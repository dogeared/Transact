Transact

Download this project as a .zip file Download this project as a tar.gz file

This is the iPhone App source code

It's written using Titanium Studio

The purpose of the Transact App is to record monetary transactions.

There are 3 tabs:

  1. Transact: For entering transactions
  2. History: Shows transaction history ordered by date
  3. Settings

This project goes along with its server side counterpart: DateABank.

You can read more about the project at my blog, afitnerd.
